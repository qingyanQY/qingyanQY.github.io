Hello Minecraft! Launcher (简称 HMCL) 是 huanghongxun 开发的一款可在多平台启动 Minecraft Java Edition 的游戏启动器。可启动范围不包含我的世界中国版。开源软件

主要功能


多游戏目录同时管理
只需打开一个启动器，即可按需启动安装在电脑硬盘中不同位置的游戏
自动式安装
在线获取所有游戏版本，为所安装的游戏核心提供全自动配置安装 Forge、Liteloader、Optifine 功能以及资源文件补齐功能。
个性化配色
多种启动器界面配色方案，提供更舒适的视觉体验
多种便捷功能
正版登录、输出含警告与错误信息的游戏日志、生成一键启动游戏的脚本、一键清理游戏目录、更新整合包、为游戏补齐缺失库与资源文件、一键删除游戏版本文件、便捷式 Mod 管理、更改运行游戏的 Java 版本、更改游戏窗口大小… 一切为玩家考虑。
